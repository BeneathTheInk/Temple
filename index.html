A lightweight JavaScript view framework for building first class web interfaces.

Download Temple Latest v0.3.6

Lightweight

Web interfaces can be highly complex and irregular, so Temple has been kept small and focused to help it work in a wide variety of situations. With no external dependencies, Temple weighs in at under 30KB minified.

Modular

In Temple, views are encapsulated, reusable components making interfaces easier to test and maintain. Only the foundation for an interactive UI is provided, allowing Temple to be a flexible base for any web application.

Reactive

Reactivity in Temple is powered by Meteor's dependency package. These auto-running computations keep interfaces up-to-date consistently and effeciently without getting in the way of existing data models.
