use

view.use( fn [, args... ] )

Temple instances can be extended through plugin functions. The use() method simply invokes fn with a context of the Temple instance as well as any args as arguments. This method gives plugins full control over the specific instance, keeping it focused and contained. If you need all views to use a plugin, extend Temple and call use() in an initialize method.
