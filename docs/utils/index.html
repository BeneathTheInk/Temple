Utilities & Extras

extend

Temple.extend( protoMethods [, classMethods ] )

To create a subclass of Temple, you can extend it by providing instance methods to add to the prototype of the subclass as well as optional class properties that are attached directly to the constructor. Extend will properly set up the prototype chain, so subclasses will always be instanceof Temple and can be further extended to any depth.

var View = Temple.extend({
    initialize: function() {}
});

var SubView = View.extend({
    customMethod: function() {}
});

var myView = new SubView;
myView instanceof View; // true
myView instanceof Temple; // true

Extend in Temple is taken directly from Backbone, so the same rules apply. Please see the Backbone documentation on extend for more details.
